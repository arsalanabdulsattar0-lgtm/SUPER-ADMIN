# Custom Rules for Coding Assistant

## UI Elements Cursor Pointer Rule
- **Date Inputs**: Whenever rendering or adding standard or custom date picker inputs (e.g. `<input type="date">` or date selection fields), always make sure that hovering over the input element or its calendar selector indicator triggers a pointer finger cursor (`cursor: pointer`).
- **Select / Dropdown Elements**: Whenever rendering or adding select inputs or custom select dropdown selectors, always ensure they show a pointer finger cursor (`cursor: pointer`) on hover.
- **Toggle Switches / Interactive Widgets**: Ensure all interactive toggles, checkboxes, and buttons have the pointer cursor state to give clear visual feedback of clickability.
